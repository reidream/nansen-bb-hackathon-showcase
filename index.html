    監視・分析

    moltbot

    Solanaミーム監視・分析ダッシュボード型

    作成者: @bb_mol_crypto

    Solanaミームは動きが速く、気づいた時にはBUYが走った後になりやすい。moltbotは通知と公開ダッシュボードで、根拠確認と追跡判断までを短くするBotです。

    GitHubで見る ダッシュボードを見る デモを見る

    目的

    急なBUY通知を起点に、Smart Moneyの根拠と公開ダッシュボード上の追跡優先度を一気に確認するためのBotです。

    • 検知する BUY速報、群衆ブレイク、whale BUYを先に拾う
    • 確認する smart-money/dex-tradesやToken Screener由来の根拠を見る
    • 判断する 公開ダッシュボードで候補を追うか見送るか決める

    できること

    • オンデマンド銘柄分析
    • Smart Money監視
    • BUY速報
    • 時系列ダイジェスト
    • 候補トークン比較
    • 公開ダッシュボード

    Nansen API / Signals

    API: Nansen API API: smart-money/dex-trades API: Token Screener API: Nansen Smart Wallets API: token analysis Signal: BUY速報 Signal: Smart Money監視 Signal: 時系列ダイジェスト Signal: 群衆ブレイク Signal: whale BUY Signal: 候補トークン追跡優先度

    見どころ

    • Nansen は token analysis、Token Screener、smart-money/dex-trades の根拠確認に絞る
    • BUY通知は群衆ブレイクやwhale BUYを先に知らせ、詳細はダッシュボードで追える
    • sm_signal_events を期間別に集計し、通知に出ない低シグナル候補も含めて見られる

    使い方

    1. 1

      通知で変化を受け取る

      BUY速報、Smart Money監視、時系列ダイジェストで、今見るべきSolanaミーム候補を把握します。

    2. 2

      根拠を確認する

      smart-money/dex-trades、Token Screener、token analysis由来の変化を候補ごとに確認します。

    3. 3

      ダッシュボードで追う

      公開ダッシュボードで群衆ブレイクやwhale BUY候補を見比べ、追跡優先度を決めます。

    AlphaCat🐱

    Smart Money買い集めアラート

    作成者: @kumicrypto

    Smart Money の買い集めは、単発の取引だけ見ても判断しにくい。AlphaCat は smart-money/dex-trades を定期確認し、複数ウォレットが同じアルトを買い始めた動きを Alpha Score で整理する。通知から DexScreener や X 検索へ進み、深掘りする候補を決めたい時に使う。

    GitHubで見るデモを見る

    目的

    複数Smart Moneyの同時買いをスコア化し、通知から深掘り対象を素早く絞るためのBotです。

    • 検知する複数ウォレットが同じアルトを買い始めた動きをまとめる
    • 評価するAlpha Score、買い集め加速、Daily TOP5で候補を並べる
    • 判断するDexScreenerやX検索リンクから深掘り候補を確認する

    できること

    • Smart Money DEX取引監視
    • Alpha Score
    • Accumulation Alert
    • エスカレーション通知
    • Daily TOP5 digest
    • DexScreener / X 検索リンク

    Nansen API / Signals

    API: smart-money/dex-tradesSignal: Smart Money DEX tradesSignal: 複数ウォレット買い集めSignal: Alpha ScoreSignal: 買い集め加速Signal: Daily TOP5

    見どころ

    • Nansen利用はランタイムで確認できる smart-money/dex-trades に絞って説明する。
    • 24時間のrolling activityとユニークウォレット数で同一トークンの買い集めを見る。
    • 通常通知、エスカレーション通知、Daily TOP5 digestで優先度を分ける。

    使い方

    1. 1

      買い集め通知を見る

      複数のSmart Moneyウォレットが同じアルトを買い始めた通知を受け取る。

    2. 2

      重なりを確認する

      Alpha Score、ユニークウォレット数、買い集め加速で候補の強さを読む。

    3. 3

      詳細リンクへ進む

      DexScreener や X 検索リンクから、追う候補をさらに確認する。

    • 確認する24h SM動向、ホルダー数、ネットフロー、Polymarket文脈を読む
    • 判断する4色判定、理由、Nansen詳細リンクで深掘りするか決める

    • SMホルダー数表示

    • Polymarketナラティブ文脈
    • 4色判定と理由
    • Nansen詳細リンク
